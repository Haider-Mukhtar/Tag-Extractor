## Run Locally

Clone the project

```bash
  git clone https://github.com/Haider-Mukhtar/Tag-Extractor.git
```

Go to the project directory

```bash
  cd Tag-Extractor
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
